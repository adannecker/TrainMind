# Branding Assets

Dieser Ordner ist fuer alle TrainMind Brand-Assets.

## Struktur

- `logo/`: Logo-Varianten (SVG/PNG)
- `icon/`: App-Icons
- `social/`: Social- und OG-Bilder
- `splash/`: Splash-/Launch-Screens
- `exports/`: exportierte Endformate fuer Deployment
